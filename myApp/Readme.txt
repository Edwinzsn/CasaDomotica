Nombre del Proyecto: CasaDomotica

Descripción:
CasaDomotica es una innovadora aplicación para Android diseñada para facilitar 
la gestión de dispositivos inteligentes en el hogar. Con una interfaz intuitiva 
y fácil de usar, permite a los usuarios controlar luces, regular la temperatura, 
gestionar la seguridad y supervisar otros dispositivos conectados desde su 
teléfono móvil. La aplicación ofrece una experiencia eficiente y cómoda para 
optimizar el control del hogar inteligente.
